## Criar cobrança

Use `/checkouts/create` para criar uma cobrança com checkout integrado. Você fornece os itens que deseja vender, e nós geramos automaticamente uma URL onde o cliente concluirá o pagamento.

<Card title="Campos obrigatórios" icon="info" horizontal>
  Apenas `items` é obrigatório. O valor total é calculado automaticamente a partir dos itens fornecidos.
</Card>

**Exemplo:**
```json
POST /checkouts/create
{
  "items": [                                    // obrigatório
    {
      "id": "prod_456",                         // ID do produto na sua loja
      "quantity": 2                             // quantidade
    }
  ],
  "customerId": "cust_abc123xyz",              // opcional - ID do cliente já cadastrado
  "customer": {                                 // opcional - dados do cliente (todos obrigatórios se usar)
    "name": "João Silva",
    "email": "customer@example.com",
    "cellphone": "+5511999999999",
    "taxId": "12345678900"
  },
  "externalId": "pedido-123",                  // opcional - ID no seu sistema
  "returnUrl": "https://seusite.com/voltar",   // opcional - URL de retorno
  "completionUrl": "https://seusite.com/sucesso", // opcional - URL após pagamento
  "metadata": {                                 // opcional - dados adicionais
    "customField": "value"
  }
}
```

**Resposta:**
```json
{
  "data": {
    "id": "bill_abc123xyz",
    "externalId": "pedido-123",
    "url": "https://app.abacatepay.com/pay/bill_abc123xyz",
    "amount": 10000,
    "paidAmount": null,
    "items": [
      {
        "id": "prod_456",
        "quantity": 2
      }
    ],
    "status": "PENDING",
    "coupons": [],
    "devMode": false,
    "customerId": null,
    "returnUrl": null,
    "completionUrl": null,
    "receiptUrl": null,
    "metadata": {},
    "createdAt": "2024-11-04T18:38:28.573Z",
    "updatedAt": "2024-11-04T18:38:28.573Z"
  },
  "error": null
}
```

Use a `url` retornada para redirecionar o cliente ao checkout e finalizar o pagamento.

---
title: Criar um link de pagamento
openapi: 'POST /payment-links/create'
---

Cria um link de pagamento que pode ser pago por múltiplos clientes — ideal para vendas em massa, rifas ou formulários de inscrição sem criar um checkout por cliente.

<Card title="Obrigatório" horizontal>
  Envie `"frequency": "MULTIPLE_PAYMENTS"` junto com `items`. O restante dos parâmetros é idêntico ao [Checkout](/pages/payment/create).
</Card>

**Exemplo:**
```json
{
  "frequency": "MULTIPLE_PAYMENTS",
  "items": [
    { "id": "prod_abc123xyz", "quantity": 1 }
  ],
  "externalId": "link-campanha-123",
  "completionUrl": "https://seusite.com/obrigado",
  "methods": ["PIX", "CARD"]
}
```

**Exemplo com multa e juros (apenas BOLETO):**
```json
{
  "frequency": "MULTIPLE_PAYMENTS",
  "methods": ["BOLETO"],
  "items": [
    { "id": "prod_abc123xyz", "quantity": 1 }
  ],
  "interest": { "value": 100 },
  "fine": { "value": 200, "type": "PERCENTAGE" }
}
```

<Tip title="Use a url retornada" horizontal>
Compartilhe `data.url` — cada cliente que acessar o link pode pagar de forma independente.
</Tip>

<Tip title="Multa e juros no boleto" horizontal>
Use `interest` e `fine` para configurar juros por atraso e multa caso o pagamento ocorra depois do vencimento. Os campos só se aplicam a `methods: ["BOLETO"]` e seguem o mesmo formato do checkout — veja a [referência completa](/pages/payment/reference#multa-e-juros-no-boleto).
</Tip>

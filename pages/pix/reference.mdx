O módulo de **PIX** permite que você envie dinheiro via PIX para qualquer chave PIX que não seja do titular da conta.

Você pode usar este recurso para enviar valores para terceiros, fornecedores, parceiros ou qualquer outra pessoa ou empresa através de suas chaves PIX.

<Note>
  **Diferença entre PIX e Saques (Payouts)**: Use **PIX** quando quiser enviar dinheiro para chaves PIX que não pertencem ao titular da conta. Use **Saques (Payouts)** quando quiser enviar dinheiro para sua própria chave PIX.
</Note>

## Estrutura

Uma transação PIX é representada em nossa API pela seguinte estrutura:

```json
{
  "data": {
    "id": "txn_abc123xyz",
    "status": "PENDING",
    "devMode": false,
    "receiptUrl": "https://app.abacatepay.com/receipt/txn_abc123xyz",
    "amount": 10000,
    "platformFee": 100,
    "externalId": "pix-123",
    "createdAt": "2024-11-04T18:38:28.573Z",
    "updatedAt": "2024-11-04T18:38:28.573Z"
  },
  "success": true,
  "error": null
}
```

## Tipos de Chave PIX

Ao enviar um PIX, você precisa especificar o tipo da chave PIX de destino:

<Info>
| Tipo      | Descrição                                                                 | Exemplo              |
| --------- | ------------------------------------------------------------------------- | -------------------- |
| `CPF`     | Chave PIX usando CPF (11 dígitos)                                         | `12345678901`        |
| `CNPJ`    | Chave PIX usando CNPJ (14 dígitos)                                        | `12345678000190`     |
| `PHONE`   | Chave PIX usando telefone celular (formato: DDD + número)                 | `11987654321`        |
| `EMAIL`   | Chave PIX usando e-mail                                                   | `contato@exemplo.com`|
| `RANDOM`  | Chave PIX aleatória (UUID)                                                | `a1b2c3d4-e5f6-...`  |
| `BR_CODE` | Chave PIX usando código BR (QR Code)                                      | Código completo do QR|
</Info>

## Limites e Taxas

- **Valor mínimo:** R$ 1,00
- **Taxa da plataforma:** Consulte nossos [termos de uso](https://abacatepay.com/termos) para informações sobre taxas
- **Processamento:** Instantâneo
- **Disponibilidade:** 24/7
- **Limite de uso:** Consulte os limites da sua conta no dashboard. Tentativas em excesso resultarão em erro HTTP 429 (Too Many Requests)

## Segurança

- Todos os PIX são autenticados via Bearer Token
- Logs detalhados de todas as transações
- Possibilidade de cancelamento em caso de erro
- Abusos desta API podem resultar em suspensão da conta conforme descrito em nossos [termos de uso](https://abacatepay.com/termos)

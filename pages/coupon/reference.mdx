Os cupons de desconto permitem que você ofereça reduções de preço aos seus clientes de forma controlada e estratégica. Você pode criar cupons com desconto percentual ou valor fixo, definir limites de uso e acompanhar quantas vezes foram utilizados.

## Criar cupom

Use `/coupons/create` para criar um cupom de desconto. Você pode definir desconto percentual ou valor fixo, limites de uso e acompanhar quantas vezes foi utilizado.

<Card title="Campos obrigatórios" icon="info" horizontal>
  Os campos `code`, `discount` e `discountKind` são obrigatórios. O `code` deve ser único e o `discountKind` pode ser `PERCENTAGE` ou `FIXED`.
</Card>

**Exemplo:**
```json
POST /coupons/create
{
  "code": "MY_COUPON",                    // obrigatório - identificador único do cupom
  "discountKind": "PERCENTAGE",           // obrigatório - tipo de desconto (PERCENTAGE ou FIXED)
  "discount": 10,                         // obrigatório - valor do desconto
  "maxRedeems": -1,                      
  "notes": "Cupom de desconto especial",  
  "metadata": {                           
    "campaign": "black-friday"
  }
}
```

**Resposta:**
```json
{
  "data": {
    "id": "MY_COUPON",
    "discountKind": "PERCENTAGE",
    "discount": 10,
    "maxRedeems": -1,
    "redeemsCount": 0,
    "status": "ACTIVE",
    "devMode": true,
    "notes": "Cupom de desconto especial",
    "createdAt": "2025-01-23T14:06:16.880Z",
    "updatedAt": "2025-01-23T14:06:16.880Z"
  },
  "error": null
}
```

Para receber notificações de atualização, você precisa cadastrar um webhook.

## <Icon icon="mobile" type="solid"/>  Gerenciando webhooks

Você pode listar, criar e remover seus webhooks dentro da nossa plataforma.

<Card title="Webhooks funcionam em ambientes diferentes." icon="book" horizontal>
  Se você criar um webhook em dev mode, você receberá notificações de atualizações no ambiente de testes. Se você criar um webhooks em produção, ele funcionará com dados reais. <br />
  Leia mais sobre dev mode <a href="./devmode"> aqui </a>.
</Card>

## <Icon icon="key" type="solid"/>  Criando webhooks

<Card title="Ao criar seu webhook, lembre-se de protegê-lo." icon="light-emergency-on" horizontal>
 • Utilize um secret em seu webhook.
</Card>

<br />

## <Icon icon="lock" type="solid"/> Protegendo requisições

Enviaremos o `secret` definido no passo anterior como query string.

Considere que esse seja seu webhook:

• `https://meusite.com/webhook/abacatepay`

Enviaremos uma requisição para o URL definido contendo o secret no final como query string:

• `https://meusite.com/webhook/abacatepay?webhookSecret=123`

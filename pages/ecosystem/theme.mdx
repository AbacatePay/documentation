## O que é o AbacatePay Theme?

O **AbacatePay Theme** é o **tema oficial da AbacatePay**, projetado para oferecer uma experiência visual consistente, confortável e produtiva em **ambientes de desenvolvimento modernos**.

Ele segue uma abordagem **green-first**, alinhada à identidade visual da AbacatePay, com foco em:

- Leitura prolongada sem fadiga
- Contraste equilibrado
- Destaque semântico de código
- Consistência entre editores

<Card title="Quando usar o AbacatePay Theme?" icon="help-circle" horizontal>
  * Você quer **consistência visual** entre ferramentas
  * Trabalha longas horas no editor
  * Valoriza **identidade visual forte** no setup
  * Prefere temas limpos, modernos e sem excesso de cor
</Card>

## Instalação

### VS Code

Instale diretamente pelo Marketplace:

```bash
code --install-extension abacatepay.abacatepay-theme
```

Ou acesse:

[marketplace.visualstudio.com/items?itemName=abacatepay.abacatepay-theme](https://marketplace.visualstudio.com/items?itemName=abacatepay.abacatepay-theme)

**Temas disponíveis**

- AbacatePay Dark Theme
- AbacatePay Light Theme
- AbacatePay Soft Theme

E também temos suporte para temas tem *itálico*

- AbacatePay Dark *Italic* Theme
- AbacatePay Soft *Italic* Theme
- AbacatePay Light *Italic* Theme**

#### Sincronizar com o sistema

Você pode sincronizar facilmente entre seus temas preferidos dark / light mode dentro do VS Code.

```jsonc
{
    "window.autoDetectColorScheme": true,
    "workbench.preferredDarkColorTheme": "AbacatePay Dark Theme",
    "workbench.preferredLightColorTheme": "AbacatePay Light Theme"
}
```

O JSON acima deve ficar no arquivo `settings.json` do seu VS Code.

### IntelliJ / JetBrains IDEs

O tema é distribuído como Color Scheme nativo.

#### Como instalar

<Steps>
    <Step title="Intellij">
        Entre na sua IDE JetBrains que você deseja configurar o tema
    </Step>
    <Step title="Configurações">
        Vá em Settings / Preferences → Editor → Color Scheme → Clique no ⚙️ → Import Scheme
    </Step>
    <Step title="Escolhendo o tema">
        Selecione um dos arquivos abaixo que deseja e aplique-o
    </Step>
</Steps>

```
intellij/abacatepay-theme-dark.json
intellij/abacatepay-theme-light.json
intellij/abacatepay-theme-soft.json
```

<Card title="Sem plugins" icon="check-circle"> Funciona nativamente em IDEs JetBrains, sem dependências externas. </Card>

### Neovim

Tema oficial com suporte a múltiplas variações.

**lazy.nvim**

```nvim
{
    "AbacatePay/vscode-theme",
    lazy = false,
    priority = 1000,
    config = function()
        vim.cmd.colorscheme("abacatepay-theme")
    end,
}
```

**packer.nvim**

```nvim
use({
    "AbacatePay/vscode-theme",
        config = function()
        vim.cmd.colorscheme("abacatepay-theme")
    end,
})
```

**Variações**

```lua
vim.cmd("colorscheme abacatepay-theme")       -- Dark (default)
vim.cmd("colorscheme abacatepay-theme-light") -- Light
vim.cmd("colorscheme abacatepay-theme-soft")  -- Soft
```

**Lualine**

```lua
require("lualine").setup {
    options = {
        theme = "abacatepay-theme"
    }
}
```

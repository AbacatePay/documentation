<Tip>
  **Para autenticar com nossa API você precisa de uma chave de API**: É com essa chave que nossa API identifica a sua conta e te permite enviar requisições com sucesso.
</Tip>


## <Icon icon="lock" type="solid"/>  Gerenciando chaves de API
Você pode listar, criar e invalidar suas chaves de API dentro da nossa plataforma.

<Card title="Chaves de API funcionam em ambientes diferentes." icon="book" horizontal>
  Se você criar uma chave de API em dev mode, ela funcionará no ambiente de testes. Se você criar uma chave de API em produção, ela funcionará com dados reais. <br />
  Leia mais sobre dev mode <a href="./devmode"> aqui </a>.
</Card>

Caso a sua chave de API não seja informada no header, esteja inválida ou revogada, nossa API retornará o código HTTP `401`.


## <Icon icon="key" type="solid"/>  Criando chaves de API

<Card title="Ao criar sua chave de API, lembre-se de armazená-la em um local seguro." icon="light-emergency-on" horizontal>
  * • Não compartilhe suas chaves de API de forma a comprometê-las.
  * • Nós não pedimos as suas chaves de API.
  * • Você pode revogar uma chave sempre que quiser, dentro da nossa plataforma.
</Card>

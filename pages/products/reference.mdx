Os produtos permitem que você crie e gerencie itens que podem ser utilizados nas suas cobranças. Com os produtos, você pode definir preços, descrições e organizar seu catálogo de forma centralizada.

## Criar produto

Use `/products/create` para criar um novo produto. Você pode definir nome, preço, moeda e descrição opcional do produto.

<Card title="Campos obrigatórios" icon="info" horizontal>
  Os campos `externalId`, `name`, `price` e `currency` são obrigatórios. O `externalId` deve ser único no seu sistema.
</Card>

**Exemplo:**
```json
POST /products/create
{
  "externalId": "prod-123",              // obrigatório - identificador único do produto no seu sistema
  "name": "Produto Exemplo",             // obrigatório - nome do produto
  "price": 10000,                        // obrigatório - preço em centavos
  "currency": "BRL",                     // obrigatório - moeda do produto
  "description": "Descrição do produto"  // opcional - descrição do produto
}
```

**Resposta:**
```json
{
  "data": {
    "id": "prod_abc123xyz",
    "externalId": "prod-123",
    "name": "Produto Exemplo",
    "description": "Descrição do produto",
    "price": 10000,
    "devMode": false,
    "currency": "BRL",
    "createdAt": "2024-11-04T18:38:28.573Z",
    "updatedAt": "2024-11-04T18:38:28.573Z",
    "status": "ACTIVE"
  },
  "error": null
}
```

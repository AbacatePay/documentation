## Estrutura

Um payout é representado em nossa API pela seguinte estrutura de transação:

```json
{
  "data": {
    "id": "txn_abc123xyz",
    "status": "PENDING",
    "devMode": false,
    "receiptUrl": null,
    "amount": 10000,
    "platformFee": 100,
    "externalId": "saque-123",
    "createdAt": "2024-11-04T18:38:28.573Z",
    "updatedAt": "2024-11-04T18:38:28.573Z"
  },
  "success": true,
  "error": null
}
```

## Atributos:

<AccordionGroup>
    <Accordion title="id:">
        ```json json {2}
        {
        "id": "txn_abc123xyz",
        ```

        `id` : <u> string. </u> <br />

    <Accordion title="status:">

        <Info >
            |                      |                                                                                                      |
            | ---------------------| ---------------------------------------------------------------------------------------------------- |
            | `PENDING`            | **Payout criado e aguardando processamento**                                                          |
            | `EXPIRED`            | **Payout expirado e não pode mais ser processado**                                                    |
            | `CANCELLED`          | **Payout cancelado**                                                                                 |
            | `COMPLETE`           | **Payout processado e concluído com sucesso**                                                        |
            | `REFUNDED`           | **Payout reembolsado**                                                                               |
        </Info>
    </Accordion>
    <Accordion title="devMode:">
        ```json json {2}
        {
        "devMode": false,
        }
        ```
        `devMode` : <u> boolean. </u> <br />
        Indica se o payout foi criado em ambiente de desenvolvimento (sandbox) ou produção.
    </Accordion>
    <Accordion title="receiptUrl:">
        ```json json {2}
        {
        "receiptUrl": null,
        ```

        `receiptUrl` : <u> string (nullable). </u> <br />

    </Accordion>
    <Accordion title="amount:">

        ```json json {2}
        {
        ```

## Limites e Taxas

- **Valor mínimo:** R$ 3,50 (350 centavos)
- **Taxa da plataforma:** R$ 0,80 centavos se estiver dentro dos limites descritos em nossos [termos de uso](https://abacatepay.com/termos)
- **Processamento:** Instantâneo
- **Disponibilidade:** 24/7 caso os limites diários e noturnos da sua conta não tenham sido excedidos
- **Limite de uso:** 1 payout por minuto. Tentativas em excesso resultarão em erro HTTP 429 (Too Many Requests). Para aumentar o limite entre em contato com nosso suporte

## Segurança

- Todos os payouts são autenticados via Bearer Token
- Logs detalhados de todas as transações
- Possibilidade de cancelamento em caso de erro
- Abusos desta API podem resultar em suspensão da conta conforme descrito em nossos [termos de uso](https://abacatepay.com/termos)

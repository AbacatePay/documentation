Quando você cria a sua conta na AbacatePay, automaticamente você está em dev mode. Mas o que isso significa?
Dev mode quer dizer que todas as interações dentro plataforma, rodam em ambiente de desenvolvimento e todos os dados são simulados.

Você pode criar chaves de API para testar em dev mode de forma sossegada, sem mexer com dinheiro de verdade.

Quando você tiver certeza de que já está com tudo certo para ir pra produção, você pode desligar o dev mode, finalizar seu cadastro e aguardar umas horinhas para começar a usar a plataforma ao vivo!

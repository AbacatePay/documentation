## <Icon icon="Avocado" type="solid"/>  O que é a AbacatePay?

A AbacatePay é um gateway de pagamento que nasceu após a necessidade de simplificar cobranças em nossos próprios produtos. Vimos que todos os meios de pagamentos eram complicados demais. Documentação extensa, longo processo de homologação, várias formas de fazer a mesma coisa e alguns gateways de pagamento até oferecem cursos para você conseguir integrar com eles!

Nossa plataforma auxilia você a cobrar seus clientes de forma simples e descomplicada. Pense que pegamos toda a parte difícil de uma Fintech e abstraímos pra você em uma API bem tranquila de integrar.


## <Icon icon="laptop" type="solid"/>  API descomplicada

Nossa API é baseada em intenção. Cada operação em cada endpoint é previsível pois cada URL representa exatamente o que você lê.

Recebemos JSON, retornamos JSON sempre com o mesmo formato. O que auxilia a integração independente da linguagem utilizada.

Pra facilitar mais ainda a integração, oferecemos alguns SDKs.

Também é possível você criar chaves de API para integrar em dev mode.

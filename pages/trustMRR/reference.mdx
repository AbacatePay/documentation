As rotas **Public MRR** expõem métricas de receita da loja para integração com plataformas Trust MRR ou para exibir no seu site ou dashboard.

## Estrutura

### Merchant (`/public-mrr/merchant-info`)

Dados básicos da loja:

```json
{
  "data": {
    "id": "store_abc123xyz",
    "name": "Example Tech",
    "website": "https://www.example.com",
    "createdAt": "2024-12-06T18:53:31.756Z"
  },
  "error": null
}
```

### MRR (`/public-mrr/mrr`)

Receita recorrente mensal:

```json
{
  "data": {
    "mrr": 0,
    "totalActiveSubscriptions": 0
  },
  "error": null
}
```

### Receita por período (`/public-mrr/revenue`)

Totais e transações por dia:

```json
{
  "data": {
    "totalRevenue": 150000,
    "totalTransactions": 45,
    "transactionsPerDay": {
      "2024-01-15": { "amount": 5000, "count": 3 },
      "2024-01-16": { "amount": 3000, "count": 2 }
    }
  },
  "error": null
}
```

## Atributos

**Merchant (merchant-info):**

| Atributo   | Tipo   | Descrição |
| ---------- | ------ | --------- |
| `id`       | string | ID único da loja |
| `name`     | string | Nome da loja |
| `website`  | string | Site da loja |
| `createdAt`| string | Data de criação (ISO 8601) |

**MRR (mrr):**

| Atributo                    | Tipo    | Descrição |
| -------------------------- | ------- | --------- |
| `mrr`                      | number  | Receita recorrente mensal em centavos (0 = nenhuma) |
| `totalActiveSubscriptions` | integer | Quantidade de assinaturas ativas (0 = nenhuma) |

**Receita (revenue):**

| Atributo              | Tipo   | Descrição |
| --------------------- | ------ | --------- |
| `totalRevenue`        | number | Receita total do período em centavos |
| `totalTransactions`   | integer| Número de transações no período |
| `transactionsPerDay`  | object | Por dia (chave = data YYYY-MM-DD); cada valor tem `amount` (centavos) e `count` |

<Tip title="Valores em centavos" icon="coins" horizontal>
 Valores monetários vêm em centavos. Divida por 100 para reais (ex.: 150000 = R$ 1.500,00).
</Tip>
